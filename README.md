# memuslugi-website
Вебсайт с мемами с со стилизацией под госуслуги

====================================================================

Для запуска скачайте библиотеку Flask и запустите app.py командами:

linux:
$	python3 app.py
windows:
$	py app.py

Добавления фотографий происходят простым переносом их в папку нужной категории:
- Остальные: all
- IT: it
- Коты: cats
- Школа: school
- По играм: games
